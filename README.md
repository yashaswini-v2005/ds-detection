# Digital Screening Aid for Down Syndrome Detection

This project is an AI-powered digital screening tool designed to analyze infant facial images and provide a probability score indicating the likelihood of Down Syndrome traits. The system uses computer vision techniques, a CNN deep learning model, and a user-friendly Streamlit interface to support early, non-invasive screening.

---

## 🌟 Features

- 🧠 **Deep Learning Classification (CNN-Based)**  
  Predicts Down Syndrome probability from infant facial images.

- 📸 **Image Preprocessing using OpenCV**  
  Face detection, resizing, normalization, and cropping.

- 🌐 **Interactive Streamlit Web App**  
  Allows users to upload images and view results instantly.

- 📊 **Probability Score Visualization**  
  Shows prediction confidence for transparent and interpretable results.

- ⚡ **Fast & Lightweight Implementation**  
  Optimized for real-time inference.
